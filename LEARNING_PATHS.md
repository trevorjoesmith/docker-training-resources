# Learning Paths by Role

Different roles have different Docker learning needs. Choose your path below!

## üéØ For Developers

**Goal:** Containerize your applications and use Docker in daily development

### Recommended Path:
1. **Week 1: Basics**
   - [Introduction (15 min)](https://docs.docker.com/get-started/introduction/)
   - [Docker Concepts](https://docs.docker.com/get-started/docker-concepts/)
   - [Your language guide](README.md#language-specific-guides) (20-30 min)

2. **Week 2: Building Images**
   - [Building Images Guide](https://docs.docker.com/get-started/docker-concepts/building-images/)
   - [Multi-stage builds](https://docs.docker.com/get-started/docker-concepts/building-images/multi-stage-builds/)
   - Practice: Containerize one of your projects

3. **Week 3: Multi-container Apps**
   - [Docker Compose Guide](https://docs.docker.com/guides/docker-compose/)
   - [Multi-container applications](https://docs.docker.com/get-started/docker-concepts/running-containers/multi-container-applications/)
   - Practice: Add a database to your containerized app

4. **Week 4: Advanced**
   - [Container-supported development](https://docs.docker.com/guides/container-supported-development/)
   - [Docker Build Cloud](https://docs.docker.com/guides/docker-build-cloud/)

## üîê For DevOps/Platform Engineers

**Goal:** Build and maintain container infrastructure and CI/CD pipelines

### Recommended Path:
1. **Foundation (Week 1-2)**
   - Complete the [Docker Workshop](https://docs.docker.com/get-started/workshop/)
   - [Building Images](https://docs.docker.com/get-started/docker-concepts/building-images/)
   - [Docker Compose](https://docs.docker.com/guides/docker-compose/)

2. **CI/CD Integration (Week 3)**
   - [GitHub Actions with Docker](https://docs.docker.com/guides/gha/)
   - [Azure Pipelines with Docker](https://docs.docker.com/guides/azure-pipelines/)
   - [Docker Build Cloud](https://docs.docker.com/guides/docker-build-cloud/)

3. **Orchestration (Week 4)**
   - [Deploy to Kubernetes](https://docs.docker.com/guides/kube-deploy/)
   - [Deploy to Swarm](https://docs.docker.com/guides/swarm-deploy/)

4. **Security & Best Practices (Week 5)**
   - [Docker Scout](https://docs.docker.com/guides/docker-scout/)
   - Docker Hardened Images
   - [Mastering Buildx Bake](https://docs.docker.com/guides/bake/)

### Recommended Path:
1. **Understanding Containers (Week 1)**
   - [Docker Concepts](https://docs.docker.com/get-started/docker-concepts/)

## üéì For Students/Beginners

**Goal:** Learn containers from scratch

1. **Start Here**
   - [QUICKSTART.md](QUICKSTART.md) - Get Docker running in 30 minutes

2. **Deep Dive**

3. **Practice Projects**

   - Deploy to Docker Hub
