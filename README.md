## 🚀 Self-Guided Learning

### Getting Started

Begin with these foundational guides to get you started with Docker. First, download and install Docker Desktop, then navigate through each section at your own pace.

- **[Introduction (15 minutes)](https://docs.docker.com/get-started/introduction/)** – Build and run your first container
- **[Docker Workshop](https://docs.docker.com/get-started/workshop/)** – 9-part hands-on workshop covering containerization, images, sharing apps, persistence, and Docker Compose
- **[Building Images](https://docs.docker.com/get-started/docker-concepts/building-images/)** – Learn how to build production-ready images with multi-stage build best practices

### Docker Concepts

- **What is a container?** – Understand the fundamentals of containerization
- **What is an image?** – Learn about Docker images and how they work
- **What is a registry?** – Understand Docker Hub and container registries
- **What is Docker Compose?** – Learn how to define and run multi-container applications

All concept guides available at: [docs.docker.com/get-started/docker-concepts](https://docs.docker.com/get-started/docker-concepts/)

### Docker Products & Tools

- **[Docker Compose](https://docs.docker.com/guides/docker-compose/)** – Create and run multi-container applications up to 39x faster using cloud-based resources
- **[Docker Build Cloud](https://docs.docker.com/guides/docker-build-cloud/)** – Simplify development, testing, and deployment of multi-container apps
- **[Docker Scout](https://docs.docker.com/guides/docker-scout/)** – Automate software supply chain security and vulnerability remediation
- **[Docker Admin Guide](https://docs.docker.com/guides/admin-set-up/)** – Set up your organization for security and productivity
- **Docker Hardened Images** – Ship with secure, enterprise-ready images that reduce vulnerabilities
